## 🏗️ Installation

If you're new to Flutter the first thing you'll need is to follow the setup instructions.

1. Run flutter channel dev
2. Run flutter upgrade

To start locally

1. flutter run -d android
2. flutter run -d ios
3. flutter run -d web
